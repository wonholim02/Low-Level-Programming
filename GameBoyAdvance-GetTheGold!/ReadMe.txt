Game Title: FIND THE GOLD!

Description: 
FIND THE GOLD! is a simple GBA game in which the player has to move through a maze while avoiding walls and a moster.
The objective is to reach the gold end point without touching any walls and approaching to moster.

Controls:
- D-pad UP: Move the player block up
- D-pad DOWN: Move the player block down
- D-pad LEFT: Move the player block left
- D-pad RIGHT: Move the player block right
- START: Begin the game
- SELECT(Backspace): Reset the game
